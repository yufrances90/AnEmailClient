# AnEmailClient
An email client app with the MERN stack


### To Run Application

> - `git clone https://github.com/yufrances90AnEmailClient.git`
> - `cd back-end`
> - `cd emailserver`
> - `npm install`
> - `node index`
> - Open a new terminal 
> - `cd front-end`
> - `cd emailclient`
> - `npm install`
> - `yarn start`
